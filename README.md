# Cool Name

React App for [cool-name-api](https://github.com/Naim-Bijapure/cool-name-api) (built with React Hooks).

### Development

-   Fork Repository.

-   Clone Repository.

```sh
git clone https://github.com/{Your_Username}/cool-name.git
cd cool-name
```

-   Install Dependencies

```sh
yarn
```

-   Start Dev Server

```sh
yarn start
```
